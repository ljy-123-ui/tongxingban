# 童行伴 - 儿童心理健康平台

一个专为儿童设计的心理健康测评与学习平台，帮助孩子了解自己，陪伴孩子快乐成长。

## 功能特点

- 🧠 **心理测评** - 包含压力感知、抑郁筛查、焦虑筛查、心理韧性等专业量表
- 👤 **性格测验** - 16人格分析测试，帮助了解孩子的性格特点
- 🎯 **知识竞赛** - 趣味心理知识问答，边玩边学
- 📊 **学习记录** - 跟踪孩子的心理发展轨迹和成长变化
- 🎨 **5套主题** - 宇宙蓝、宁静紫、森林绿、暮光橙、温柔粉，随心切换
- 🎵 **治愈音乐** - 温暖舒缓的背景音乐，营造放松氛围

## 技术说明

- 纯前端实现，无需服务器
- 所有资源内嵌，复制到U盘即可使用
- 数据保存在浏览器本地（localStorage）
- 响应式设计，支持电脑、平板、手机
